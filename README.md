# Python Banking System

Banking Management System developed in Python and MySQL as my CBSE Class 12 Computer Science final project. While based on the concepts taught in school, I refined and expanded the implementation by adding features such as automatic database creation, secure login, recovery key-based password reset, account management, and better input handling.

---

## Features

- Create a new bank account
- Automatic generation of a unique 10-digit account number
- Secure login with password authentication
- Password recovery using a 4-digit recovery key
- Check account balance
- Deposit money
- Withdraw money
- Update account holder's name
- Delete an account
- Automatic creation of the required MySQL database and table
- Basic input validation and error handling

---

## Technologies Used

- Python 3
- MySQL
- mysql-connector-python

## How to Run

1. Make sure MySQL Server is installed and running.
2. Clone this repository.

```bash
git clone https://github.com/Rohan-Bloch-2008/python-banking-system.git
```

3. Open the project folder.

```bash
cd python-banking-system
```

4. Install the required package.

```bash
pip install -r requirements.txt
```

5. Run the program.

```bash
python-banking-project.py
```

6. Enter your MySQL Host, Username, and Password when prompted.

The program will automatically create the required database (`banking`) and table (`project`) if they do not already exist.
